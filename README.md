# 🎓 Tech Grad Readiness Checklist

An interactive, fully customizable checklist for AI & Machine Learning graduates entering the job market. Built as a single HTML file — no installs, no accounts, no frameworks.

**[→ Open the live checklist](https://annaberriej.github.io/grad-readiness-checklist)**

---

## What's inside

50+ action items across 8 categories covering everything you need to land your first tech role:

- LinkedIn Profile
- GitHub Setup
- Portfolio Projects
- Resume
- Certifications
- Networking
- Technical Skills
- Job Search Readiness

Each item has a priority badge (Urgent / High / Medium / Low) and an explanatory note. Progress saves automatically to your browser.

---

## How to use it

Open the live link above and start checking things off. Everything saves in your browser automatically — your progress will still be there when you come back.

**To make it your own:**
- Double-click any text to edit it
- Click any badge to change the priority
- Hover an item and click ✕ to delete it
- Click **+ Add item** to add your own to any section
- Click **+ Add Section** to create a custom category
- Click **Export My Version** to download your personalized copy as an HTML file

---

## Run it locally

No setup needed. Just download `index.html` and open it in any browser.

```bash
git clone https://github.com/AnnaBerrieJ/grad-readiness-checklist.git
cd grad-readiness-checklist
open index.html
```
